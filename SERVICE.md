## ğŸ’» Tech Stack

### Core

- **Framework**: Tauri 2.9 (Rust backend + React frontend)
- **Build**: 7.1MB (vs 285MB Electron equivalent)
- **Language**: TypeScript 5.3+ (strict mode)
- **State**: Zustand (lightweight, persistent)

### AI & ML

- **LLM**: qwen2.5:7b (14.8B params, Q4_K_M quantization, 4.7GB)
- **Runtime**: Ollama (Metal acceleration on macOS)
- **Speech**: Xenova/transformers (Whisper-small)
- **Embeddings**: TF-IDF (128-dim vectors)
- **Vision**: LLaVA 7B

### Backend (Rust)

- **Database**: SQLite via rusqlite (AES-256 encrypted)
- **Async**: Tokio runtime
- **HTTP**: reqwest for Ollama API

### Frontend (React)

- **UI**: shadcn/ui (headless, accessible)
- **Styling**: Tailwind CSS
- **Markdown**: react-markdown + rehype-highlight
- **i18n**: i18next (Korean + English)

## â“ FAQ

### General

**Q: Is it really 100% private?**
A: Yes. All AI runs on your Mac via Ollama. Zero data leaves your machine unless you enable optional Google Drive backup (settings only, not conversations).

**Q: Is it really free forever?**
A: Yes. MIT License, open source, no subscriptions, no trials, no paywalls.

**Q: How does it compare to ChatGPT?**
A: Smaller model (7B vs 100B+), so less capable. But 100% private, free, offline, and customizable.

### Performance

**Q: Why are responses slower than ChatGPT?**
A: Your Mac runs a 7B model locally. ChatGPT uses massive data centers with hundreds of GPUs. 2-4s is actually quite fast for local AI!

**Q: Will it work on my 8GB Mac?**
A: Not recommended. The model needs ~12GB RAM. You'll experience severe slowdowns.

**Q: Can I use a different model?**
A: Not currently. Model selection is on the roadmap.

### Privacy

**Q: Does any data leave my computer?**
A: Only if you enable Google Cloud backup (optional). And even then, only persona settings (10 numbers), never conversations.

**Q: Can I use this completely offline?**
A: Yes! After initial setup (downloading model), works 100% offline.

**Q: What if I don't trust you?**
A: Audit the code! It's open source (MIT License). You can inspect every line.

### Technical

**Q: What languages does it support?**
A: Korean and English fully supported.

**Q: Can it browse the internet?**
A: Not yet. Internet access is planned for future release.

**Q: Can I run this on Windows?**
A: Not yet. Windows build planned for Q2 2025.
